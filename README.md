# DISE-O-APP
# App de Ejercicios

## Descripción
Aplicación enfocada en adultos de 50 a 75 años para mejorar su actividad física mediante rutinas simples y guiadas.

## Funcionalidades
- Navegación entre pantallas
- Rutinas de ejercicio
- Pantalla de confirmación de salida
- Recomendaciones de salud
- Registro de progreso

## Tecnologías
- Figma (diseño de interfaces)
- GitHub (control de versiones)

## Cómo usar
1. Revisar las pantallas del proyecto
2. Seguir el flujo de la aplicación desde login hasta completar la rutina
